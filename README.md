# Substance-Painter-custom-shader
 A custom shader made for Substance Painter for color customization system in the texture production pipeline for clothes and props. This custom shader allows 3d artists to visualize the final result and make changes inside SP. 
 
 This shader is inspired by the GDC talk "The art and technology behind creating characters for Baldur's Gate 3".
